Problem 2¶

Below are density plots of the entire fake data with various bin sizes (10x10, 100x100, 1000x1000). At bin = 10, the plot is dominated by noise, and at bin = 1000, everything is washed out. At bin = 100, the bins seem to be closest to representing the true density.

As for the plot itself, the color of each bin represents the density of data points belonging to that bin. The more data points it contains, the more yellow it becomes. The colorbar on the size can be used to match the color with the approximate amount of data points in each bin, hence knowing the density of each bin.

In [10]:
x = np.linspace(0,len(bkgd)-1,len(bkgd))
fig, ax = plt.subplots(3,1,figsize=(20,35))
h, xe, ye, img = ax[0].hist2d(x,bkgd[0:100000],10)
cbar = plt.colorbar(img, ax=ax[0])
cbar.ax.tick_params(labelsize='20')
ax[0].tick_params(labelsize=20)
